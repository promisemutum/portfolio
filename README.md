# Portfolio

Next.js App Router portfolio with a boot sequence and a masked image reveal.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Routes

- `/` redirects to `/counter`
- `/counter` boot sequence
- `/home` masked reveal using optimized images in `public/images/optimized`

## Assets

- `optimize_textures.js` regenerates optimized and low-res images from `public/images`
